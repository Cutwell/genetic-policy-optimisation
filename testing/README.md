# Custom matrices

* Custom matrices can be generated from images.
* Image is parsed 1-1 into an environment matrix.

### Customisation
1. Image must be 8bit Greyscale PNG image.
2. L values indicate pixel representation.

| Pixel value | Representation |
|:--:|:--:|
|L=0|Air|
|L=1|Wall|
|L=2|Window|
